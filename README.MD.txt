Introduction

This is a simple javascript card game. You need to concentrate on each cards When you face it up and search for two same cards to solve the puzzle. AND it calculates the seconds you consumed to solve this puzzle. Does it Grab your interest?

Rules

1. You can't face up more then two cards at a time.
2. Only one winning condition, i.e. find two same cards.
3. No option to quit the game in progress but there is always a browser close button anyways.
